# Makeshift 3D Object Design Tool
Design and compose 3D objects, experiment with various shapes and shaders, export your work to a standalone HTML file.

SourceForge page: https://sourceforge.net/projects/makeshift-3d-object-designer/

Link: https://makeshift-3d-object-design-tool.netlify.app/
